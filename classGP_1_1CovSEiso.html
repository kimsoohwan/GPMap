Detailed Description

template<typename Scalar>
class GP::CovSEiso< Scalar >

Squared exponential covariance function with isotropic distances

\[ k(\mathbf{x}, \mathbf{z}) = \sigma_f^2 \exp\left(-\frac{r^2}{2l^2}\right), \quad r = |\mathbf{x}-\mathbf{z}| \]

All covariance classes should have public static member functions as follows.

Public Static Member Functions Corresponding Covariance Functions
+CovSEiso::K \(\mathbf{K} = \mathbf{K}(\mathbf{X}, \mathbf{X}) \in \mathbb{R}^{N \times N}\)
+CovSEiso::Ks \(\mathbf{K}_* = \mathbf{K}(\mathbf{X}, \mathbf{Z}) \in \mathbb{R}^{N \times M}\)
+CovSEiso::Kss \(\mathbf{k}_{**} \in \mathbb{R}^{M \times 1}, \mathbf{k}_{**}^i = k(\mathbf{Z}_i, \mathbf{Z}_i)\) or \(\mathbf{K}_{**} = \mathbf{K}(\mathbf{Z}, \mathbf{Z}) \in \mathbb{R}^{M \times M}\)

where \(N\): the number of training data and \(M\): the number of test data given

\[ \mathbf{\Sigma} = \begin{bmatrix} \mathbf{K} & \mathbf{k}_*\\ \mathbf{k}_*^\text{T} & k_{**} \end{bmatrix} \text{, or } \mathbf{\Sigma} = \begin{bmatrix} \mathbf{K} & \mathbf{K}_*\\ \mathbf{K}_*^\text{T} & \mathbf{K}_{**} \end{bmatrix} \]

.

The public static member functions, K, Ks and Kss call a protected general member function, K(const Hyp, const MatrixConstPtr, const int) which only depends on pair-wise squared distances.

In addition, no covariance class contains any data. Instead, data are stored in data classes such as

  1. TrainingData
  2. DerivativeTrainingData
  3. TestData

inlinestatic

Self covariance matrix between the training data, K(X, X) or its partial derivative.

Note
It calls the protected general member function, CovSEiso::K(const Hyp, const MatrixConstPtr, const int) which only depends on pair-wise squared distances.

Only this function returns partial derivatives since they are used for learning hyperparameters with training data.
Parameters
[in]logHypThe log hyperparameters
  • logHyp(0) = \(\log(l)\)
  • logHyp(1) = \(\log(\sigma_f)\)
[in]trainingDataThe training data
[in]pdHypIndex(Optional) Hyperparameter index for partial derivatives
  • pdHypIndex = -1: return \(\mathbf{K}(\mathbf{X}, \mathbf{X})\) (default)
  • pdHypIndex = 0: return \(\frac{\partial \mathbf{K}}{\partial \log(l)}\)
  • pdHypIndex = 1: return \(\frac{\partial \mathbf{K}}{\partial \log(\sigma_f)}\)
Returns
An NxN matrix pointer
N: The number of training data
