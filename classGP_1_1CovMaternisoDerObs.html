Detailed Description

template<typename Scalar>
class GP::CovMaternisoDerObs< Scalar >

Matern covariance function dealing with derivative observations
It inherits from CovDerObs which takes CovMaternisoDerObsBase as a template parameter.
Thus, CovMaternisoDerObs is a combination of CovDerObs and CovMaternisoDerObsBase.
