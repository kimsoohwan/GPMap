Detailed Description

template<typename Scalar>
class GP::CovSEisoDerObs< Scalar >

Squared exponential covariance function dealing with derivative observations
It inherits from CovDerObs which takes CovSEisoDerObsBase as a template parameter.
Thus, CovSEisoDerObs is a combination of CovDerObs and CovSEisoDerObsBase.
