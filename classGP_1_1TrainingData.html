Detailed Description

template<typename Scalar>
class GP::TrainingData< Scalar >

A training input and output data

\[ \{(\mathbf{X}, \mathbf{y}_i)\}_{i=1}^N, \quad \mathbf{X} \in \mathbb{R}^D, \; \mathbf{y} \in \mathbb{R} \]

where \(N\): number of training data, \(D\): number of dimensions.

It also holds the pair-wise squared distances, absolute distances and differences between the training inputs which will be repeatedly used in covariance functioins.
